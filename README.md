# 📊 Калькулятор оценок

Консольное приложение на C++ для учёта и анализа оценок студентов.

## Возможности

- ➕ Добавление студентов и оценок
- 📈 Расчёт среднего балла, GPA
- 📊 Статистика: минимум, максимум, медиана
- 🔤 Конвертация в буквенную систему (A, B, C, D, F)
- 💾 Сохранение и загрузка данных из файла
- 📋 Просмотр списка всех студентов с оценками

## Сборка и запуск

### С помощью CMake

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

### С помощью Docker

Создать образ и запустить контейнер можно двумя способами: через чистый Docker или с помощью Docker Compose.

**Вариант 1. Использование Docker CLI:**
```bash
# Сборка образа
docker build -t grade-calculator .

# Запуск интерактивного контейнера 
docker run -it grade-calculator
```

**Вариант 2. Использование Docker Compose:**
```bash
# Сборка и запуск контейнера 
docker-compose up -d

# Подключение к запущенному контейнеру для взаимодействия с меню
docker-compose exec grade-calculator ./build/GradeCalculator

# Остановка контейнера
docker-compose down
```

## Линтинг и Форматирование 

Код этого проекта автоматически проверяется на соответствие стандартам (стиль Google, проверки cppcheck). Для этого используется утилита `pre-commit`.

Она автоматически запускает форматтер `clang-format` и линтер `cppcheck` при каждой попытке сделать коммит, не пропуская в репозиторий некачественный код.

**Как установить и активировать:**
1. Установите Python, если его нет в системе (он необходим для работы pre-commit).
2. Запустите в терминале:
   ```bash
   pip install pre-commit
   ```
3. Установите хуки в папке проекта:
   ```bash
   pre-commit install
   ```
4. Если хотите отформатировать все файлы сразу вручную:
   ```bash
   pre-commit run --all-files
   ```

## Система оценок

| Баллы   | Буква | GPA  |
|---------|-------|------|
| 90–100  | A     | 4.0  |
| 80–89   | B     | 3.0  |
| 70–79   | C     | 2.0  |
| 60–69   | D     | 1.0  |
| 0–59    | F     | 0.0  |
